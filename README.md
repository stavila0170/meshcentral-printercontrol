# Mesh Printer Control

MeshCentral plugin for remote Windows printer, print-job and Print Spooler management.

## Install in MeshCentral

Enable plugins in MeshCentral, restart the server, then install using this configuration URL:

```text
https://raw.githubusercontent.com/stavila0170/meshcentral-printercontrol/main/config.json
```

The files in this repository must remain at repository root. In particular, `config.json` and `printercontrol.js` must not be placed inside an additional `printercontrol` directory.
